Pipeline de Datos y Despliegue

Ingeniería de Datos | Proyecto Salud Mental Adolescente

1. Descripción del Dataset

Este proyecto implementa un flujo de trabajo ETL enfocado en la salud mental de los adolescentes. El dataset analiza factores como el uso de redes sociales, horas de sueño y rendimiento académico para identificar patrones de estrés y depresión.

2. Fuente de Datos

Los datos han sido obtenidos del archivo Teen_Mental_Health_Dataset.csv, el cual contiene registros detallados de hábitos digitales y bienestar emocional. El archivo se encuentra alojado en la carpeta /data de este repositorio.

3. Diccionario de Variables

Dimensiones y métricas clave del estudio:

Variable Descripción Tipo de Dato
age Edad del adolescente encuestado. Entero (int)
daily_social_media_hours Horas promedio al día en redes sociales. Flotante (float)
platform_usage Plataforma principal (Instagram, TikTok, Both). Texto (object)
stress_level Nivel de estrés reportado (Escala 1-10). Entero (int)
depression_label Indicador de síntomas de depresión (0: No, 1: Sí). Binario (int)

4. Arquitectura del Pipeline

  • Ingesta: Carga automatizada del CSV desde el repositorio local.
  • Transformación: Limpieza de datos y clasificación de métricas de bienestar con Pandas.
  • Orquestación: Definición de flujos de tareas mediante lógica inspirada en Airflow.
  • Visualización: Dashboard interactivo con gráficas dinámicas en Streamlit Cloud.
